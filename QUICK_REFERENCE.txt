üõ†Ô∏è TROUBLESHOOTING COMMANDS
================================================================================

CONNECTION REFUSED ERROR:
-------------------------
1. Check if blockchain is running: netstat -an | findstr 8545
2. Restart blockchain: cd src/Backend && npx hardhat node --port 8545

BAD_DATA CONTRACT ERRORS:
-------------------------
1. Stop React app (Ctrl+C in Terminal 3)
2. Redeploy contracts: cd src/Backend && npx hardhat run scripts/deployCombined.js --network localhost
3. Restart React: cd ../.. && npm start

REACT APP WON'T START:
---------------------
1. Check port 3000: netstat -an | findstr 3000
2. Clear cache: rmdir /s node_modules && npm install
3. Kill processes: taskkill /f /im node.exe
4. Restart: npm start

METAMASK ISSUES:
---------------
1. Reset account in MetaMask Advanced settings
2. Switch network to Hardhat Local (Chain ID: 31337)
3. Refresh browser page
4. Reconnect wallet

COMPLETE RESET:
--------------
1. Ctrl+C all terminals
2. cd src/Backend && npx hardhat node --port 8545
3. New terminal: cd src/Backend && npx hardhat run scripts/deployCombined.js --network localhost
4. New terminal: cd ../.. && npm start
5. Reset MetaMask account
6. Refresh browser
